# 四层发现

- 优点
  - 可路由且结果可靠
  - 不太可能被防火墙过滤
  - 甚至可以发现所有端口都被过滤的主机
- 缺点
  - 基于状态过滤的防火墙可能过滤扫描
  - 全端口扫描速度慢
- TCP
  - 未经请求的ACK-------RST
  - SYN----------SYN/ACK、RST
- UDP
  - ICMP端口不可达、一去不复返

## scapy

### TCP

```bash
scapy
i=IP()
t=TCP()
r=(i/t)
r[IP].dst="192.168.1.1"
r[TCP].flags="A"
a=sr1(r)
```

### UDP

```bash
scapy
i=IP()
u=UDP()
r=(i/u)
r[IP].dst="192.168.1.1"
r[UDP].dport=666
a=sr1(r)
```
